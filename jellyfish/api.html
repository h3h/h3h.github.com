FB.ui

Method for triggering UI interaction with Facebook as iframe dialogs or popups, like publishing to the stream, sharing links.

Parameters

Name Type Description
params Object

The required arguments vary based on the method being used, but specifying the method itself is mandatory. If display is not specified, then iframe dialogs will be used when possible, and popups otherwise.

cb Function

Optional callback function to handle the result. Not all methods may have a response.
